📌 How to use this dashboard

This dashboard loads your pipeline outputs from the outputs/ and changelog/ directories. Since it runs locally from the file system, you need to either:

Option A (Recommended): Run python scripts/serve_dashboard.py — starts a local server at http://localhost:8080 that serves JSON files.

Option B: Paste JSON content directly into the panels below using the import buttons.

Or just open this file after running python scripts/run_all.py and use the JSON import feature.
